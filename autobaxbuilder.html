TL;DR: We present AutoBaxBuilder, an automated framework that generates code security benchmark tasks from scratch, reducing manual effort by ~12Ã— while successfully reproducing or outmatching expert-written tests and exploits.

Key Takeaways

  • AutoBaxBuilder generates complete benchmark tasks (scenarios, tests, exploits) in under 2 hours for less than USD 10, representing a ~12Ã— reduction in effort compared to manual creation.
  • Generated tests and exploits successfully reproduce or outmatch expert-written ones on BaxBench scenarios, tightening the upper security bound.
  • The framework addresses benchmark contamination and enables continuous expansion with varying difficulty levels to challenge increasingly capable LLMs.

How does AutoBaxBuilder work?

Figure 1: Overview of our method. The LLM-based pipeline starts from scratch and produces a complete benchmark instance with (1) scenario description, (2) test cases and (3) end-to-end exploits. After generating a novel scenario description, the LLM generates functional tests and solutions, iterating until execution feedback confirms that the tests are solvable. Next, the LLM designs end-to-end exploits to expose vulnerabilities, iterating until it finds a pair of solutions, one on which the exploit succeeds and one on which it fails. The results are combined into a new task instance.

The AutoBaxBuilder pipeline employs an agentic LLM-based approach that starts from scratch and produces complete benchmark instances. The pipeline generates novel scenario descriptions, automatically creates functional tests through iterative refinement based on execution feedback, and designs end-to-end security exploits that are validated against multiple solutions to ensure they correctly identify vulnerabilities.

AutoBaxBuilder vs Expert-Written Tests

Comparison of AutoBaxBuilder-generated tests and exploits against expert-written ones on the original BaxBench scenarios.

To validate the quality of AutoBaxBuilder, we compared its generated tests and exploits against those written by security experts on the same BaxBench scenarios. Our evaluation shows that AutoBaxBuilder successfully reproduces or outmatches the expert-written functional tests and exploits, tightening the upper security bound reported by BaxBench.

How can I contribute to AutoBaxBench?

We welcome contributions to improve the AutoBaxBuilder pipeline, add new scenarios generated by the system, or provide feedback on the quality of generated tasks. Please visit our GitHub repository for details.
